# Switch

Multipurpose switch script with plugin infrastructure

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'switch', '~> 0.1.0', git: 'https://github.com/lscheidler/switch'
```

And then execute:

    $ bundle --binstubs bin
